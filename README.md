
# Access Testing

## วัตถุประสงค์
```
- ใช้สำหรับทดสอบอ่านไฟล์ Access ด้วย Node.js
- ไฟล์ .mdb เป็นไฟล์ Access รุ่นเก่า
- ไฟล์ .accdb เป็นไฟล์ Access รุ่นใหม่
```

## การติดตั้ง
```bash
git clone https://github.com/wasankds/access-testing.git          
cd access-testing
npm install
```
